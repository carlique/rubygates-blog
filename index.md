# Rubygates

*A game idea from 2008. Rebuilt with AI in 2026.*

---

Rubygates started as an ambitious experiment:  
a massive multiplayer, cooperative world running in the browser — long before today’s technologies made it practical.

The project was never truly finished.  
It was never deleted either.

For years, it sat archived — designs, assets, story, mechanics — waiting.

---

## Why now?

Because the tools have changed.

AI can now write code.  
Web technologies can handle real-time worlds.  
Infrastructure that once required clusters can now run in containers.

The question is no longer *“is it possible?”*  
The question is:

> **How far can AI take a real game project?**

---

## What this is

This is not just a game.

This is an experiment:
- Can AI act as a development team?
- Can a complex system emerge from prompts and iteration?
- Where are the limits — really?

I act as:
- stakeholder  
- product owner  
- tester  

AI does the rest.
